若是的近义词
近义词：
如果、倘使、倘若、假使、假如、假若、要是
如果如果 rúguǒ [if] 假如,假使 你如果要来,请事先告诉我
倘使倘使 tǎngshǐ [if;in case;provided that;in the event of]。用在偏正复句的偏句中,表示假设关系,相当于“倘若”、“假使”。多用于书面语 倘使错过这个机会,你会后悔的
倘若倘若 tǎngruò [if;in case;provided that;in the event of]。用在偏正复句的偏句中,表示假设关系,相当于“如果”、“假使”。多用于书面语 你倘若有困难,我定设法相助
假使假使 jiǎshǐ [if;suppose;in case;in the event that] 提出一种假设,相当于“如果”、“假如” 假使你遇见这桩事情,你怎么办?
假如假如 jiǎrú [if;suppose;in case;in the event that] 假使,如果 假如没有汽车,我们就不能按时赶到工地
假若假若 jiǎruò [provided;suppose;in case;in the event that] 假如 假若这些推论是合乎逻辑的话
要是要是 yàoshi [in case;if] ——用在偏正复句的偏句中,表示假设,相当于“如果”,正句有时用“就”跟它呼应 你要是没有时间来的话,我可以去找你

若是词典解释 (详细解释)
  • 词语拼音：ruò shì
  • 词语解释：

    若是 ruòshì

    [if] ——一般用在偏正复句中偏句的开头,表示假设,相当于“要是”、“如果”,正句中常用“就”、“便”跟它相呼应

    若是似都管看待我们时,并不敢怨怅。——《水浒传》

    他若是不去,我们就让别人去
